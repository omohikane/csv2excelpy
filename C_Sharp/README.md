# C_Sharp

csv2excelpyのC#バージョン

## Requirement

- Visual Studio 2019  
- ClosedXML  

```
.
├── csvToExcelFile
│   ├── Form1.cs
│   ├── bin
│ 　　   ├── Debug
│     　　   ├── csvToExcelFile.exe
│     　　   ├── csv
│     　　   ├── xlsx
├── packages
|   ├── 色々必要な
├── csvToExcelFile.sln
└── README.md
```

## Installation

PMコンソールからインストール
`Install-Package ClosedXML`

## Usage
試す場合  
`./csvToExcelFile/bin/Debug/`の中のCSVファイルをxlsxに
`./csvToExcelFile/bin/Debug/csvToExcelFile.exe`を使う
